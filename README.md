# Desafio Python + Supabase + Z-API

Projeto simples em Python que busca contatos cadastrados no Supabase e envia uma mensagem personalizada pelo WhatsApp usando Z-API.

Mensagem enviada:

```text
Olá, <nome_contato> tudo bem com você?
```

## Setup da tabela no Supabase

No painel do Supabase, abra **SQL Editor** e rode o script abaixo:

```sql
create table if not exists contacts (
  id bigint generated by default as identity primary key,
  name text not null,
  phone text not null,
  created_at timestamp with time zone default now()
);

alter table contacts enable row level security;

drop policy if exists "Allow public read contacts" on contacts;

create policy "Allow public read contacts"
on contacts
for select
to anon
using (true);

insert into contacts (name, phone) values
  ('Marcelo', '5511999999999'),
  ('Ana', '5511988888888'),
  ('Joao', '5511977777777');
```

O mesmo script esta no arquivo `supabase_setup.sql`.

Use os telefones no formato aceito pela Z-API, normalmente com DDI + DDD + numero, sem espacos ou caracteres especiais.

## Variaveis de ambiente

Copie o arquivo `.env.example` para `.env` e preencha os valores:

```env
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_KEY=sua_anon_key
SUPABASE_TABLE=contacts
CONTACT_NAME_COLUMN=name
CONTACT_PHONE_COLUMN=phone
CONTACT_LIMIT=3

ZAPI_INSTANCE_ID=sua_instance_id
ZAPI_TOKEN=seu_token
ZAPI_CLIENT_TOKEN=seu_client_token_opcional

LOG_LEVEL=INFO
```

Observacao: `CONTACT_LIMIT` sempre sera limitado a no maximo 3 envios, conforme a regra do desafio.

## Como rodar

Crie e ative um ambiente virtual:

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Instale as dependencias:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python main.py
```

## Fluxo

1. O script carrega as variaveis do `.env`.
2. Busca ate 3 contatos na tabela do Supabase.
3. Monta a mensagem com o nome do contato.
4. Envia a mensagem pela Z-API.
5. Registra sucessos e erros no terminal.
